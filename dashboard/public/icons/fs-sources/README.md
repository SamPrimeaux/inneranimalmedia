# File source icons (Simple Icons)

Brand marks for AgentSamFilesystem v2 source rail. Display at **20×20px** inside a **44×44px** hit target.

| File | Source | Tab |
|------|--------|-----|
| `local.svg` | IAM folder mark (not on Simple Icons) | Local native folder |
| `react.svg` | [React](https://simpleicons.org/?q=react) `#61DAFB` | Workspace React repo |
| `r2.svg` | [Cloudflare](https://simpleicons.org/?q=cloudflare) `#F38020` | R2 / object storage |
| `github.svg` | [GitHub](https://simpleicons.org/?q=github) `#FFFFFF` (dark UI) | GitHub |
| `googledrive.svg` | [Google Drive](https://simpleicons.org/?q=google+drive) `#4285F4` | Drive |
| `container.svg` | [Cloudflare Workers](https://simpleicons.org/?q=cloudflare+workers) `#F38020` | Sandbox / container |

Icons from [Simple Icons](https://simpleicons.org/) are [CC0 1.0](https://github.com/simple-icons/simple-icons/blob/develop/LICENSE.md).

Refresh from CDN:

```bash
curl -fsSL "https://cdn.simpleicons.org/react/61DAFB" -o react.svg
curl -fsSL "https://cdn.simpleicons.org/github/ffffff" -o github.svg
curl -fsSL "https://cdn.simpleicons.org/googledrive/4285F4" -o googledrive.svg
curl -fsSL "https://cdn.simpleicons.org/cloudflare/F38020" -o r2.svg
curl -fsSL "https://cdn.simpleicons.org/cloudflareworkers/F38020" -o container.svg
```
