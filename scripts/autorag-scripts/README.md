# inneranimalmedia-autorag/scripts/

Executable operator scripts for Inner Animal Media. **Canonical bytes live here** — D1 `agentsam_scripts` holds metadata only (`body=''`, `source_stored=r2:…`).

## Lanes

| Folder | Purpose | When to use |
|--------|---------|-------------|
| `deploy/` | Ship worker, frontend, R2 assets | Production deploy paths only |
| `maintenance/` | Hygiene, credentials, audits | D1/R2/Supabase operator tasks |
| `cicd/` | Migrations, guards, wrangler verify | Pre-ship validation |
| `infra/` | Cloudflare env, tunnels, VM sync | Platform credential wiring |
| `test/` | Smokes | Non-prod verification |
| `ingest/` | RAG embed/index pipelines | Knowledge indexing |
| `audit/` | Read-only inspection | Diagnostics |
| `benchmark/` | Perf probes | Optional measurement |
| `archive/` | Deprecated | Do not run — reference only |

## Sidecars

Each canonical script may have a sibling `*.meta.md` (frontmatter + invocation). AutoRAG indexes `.meta.md` for semantic discovery; the script file is the executable source.

## Machine index

See `_index/inventory.json` — generated by `./scripts/upload-agentsam-scripts-r2.sh`.

## Upload

```bash
./scripts/upload-agentsam-scripts-r2.sh
./scripts/upload-agentsam-scripts-r2.sh scripts/foo.sh:maintenance/foo.sh
```

## D1 contract

- `purpose` must match lane folder name
- `source_stored` = `r2:inneranimalmedia-autorag/scripts/{lane}/{file}`
- `script_hash` = sha256 of R2 object (drift detection)
