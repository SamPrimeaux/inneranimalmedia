## Top Recommendations

### 1. Wire identity columns everywhere
The most common NULL pattern across agentsam_* is `tenant_id` and `workspace_id` missing.
Every INSERT in the Worker should resolve and bind these at the handler level, not leave them to individual table writers.

### 2. Token counts are always 0 in agentsam_agent_run
`input_tokens`, `output_tokens`, and `cost_usd` are almost certainly 0 across the board.
The SSE stream handler needs to capture the usage block from the provider response and UPDATE the run row after completion.

### 3. Zombie column migration
