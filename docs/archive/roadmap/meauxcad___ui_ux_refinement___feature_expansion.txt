EXECUTION_STEPS:

### STEP None: Mobile responsiveness (iPhone 13 target)
STATUS: pending
TARGET: N/A
DESCRIPTION:
App.tsx panel layout: stack panels vertically on small viewports. Sidebar icons become a swipeable bottom drawer. Terminal and editor should not try to share horizontal space below 768px. Test on iPhone 13 Safari. No new dependencies — pure CSS/Tailwind breakpoints.
---
### STEP None: Glassmorphism + depth layer
STATUS: pending
TARGET: N/A
DESCRIPTION:
Sidebar and top bar: backdrop-filter blur(12px) with semi-transparent background. Resizer borders: glow on hover/active with solar-cyan (var --accent or equivalent). Panel backgrounds get subtle gradient depth. Do not apply blur to the 3D viewport or Monaco editor — only chrome/shell elements.
---
### STEP None: Micro-animations + motion polish
STATUS: pending
TARGET: N/A
DESCRIPTION:
Terminal panel slides up on open (translateY transition, 200ms ease-out). Dirty dot on tabs pulses subtly. Panel resize handles illuminate on grab. File tree items fade in on expand. Antigravity response streaming has a cursor blink. Keep all animations under 300ms — nothing that blocks work.
---
### STEP None: Excalidraw panel integration
STATUS: pending
TARGET: N/A
DESCRIPTION:
Add Excalidraw as a new panel type in the StudioSidebar (whiteboard icon). Embed @excalidraw/excalidraw React component. Panel should be resizable like the GLB viewer. Save/load state to R2 (CAD_STORAGE bucket, path: excalidraw/{session_id}.json). Wire Antigravity tool: agent can read and write Excalidraw scene JSON. No Excalidraw cloud — self-hosted only.
---
### STEP None: Playwright end-to-end test suite
STATUS: pending
TARGET: N/A
DESCRIPTION:
Set up Playwright in the MeauxCAD repo. Test matrix: editor loads, terminal opens, GLB viewer renders, Excalidraw panel opens, Antigravity sends a message and gets a response. Run suite in CI on every push to main. Pipe results into cidi_run_results (test_type=route). Gate prod deploy on suite passing.
---
### STEP None: 3D modeling pipeline refinement
STATUS: pending
TARGET: N/A
DESCRIPTION:
GLB viewer: add orbit controls (if not already), environment lighting toggle, wireframe mode. Add model-viewer web component as a second viewer option for AR/mobile. Upload pipeline: drag GLB into editor area → auto-upload to R2 CAD_STORAGE → open in viewer. Antigravity tool: agent can describe model metadata, suggest transformations. No Blender wasm yet — scoped to viewing + metadata.
---

TARGET: N/A
DESCRIPTION:
Allow panels to be detached, reordered, or pinned. Save layout state to D1 (user_preferences or cad-specific table). Restore layout on reload. Add a "Reset layout" button. This is the foundation for repurposing as IAM dashboard shell.
