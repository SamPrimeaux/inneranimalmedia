### STEP None: workspace_edit_file — str_replace surgical editing (no more sed disasters)
STATUS: pending
TARGET: N/A
DESCRIPTION:
Add workspace_edit_file tool. Parameters: path, old_str, new_str. CRITICAL RULES: (1) Fail with error if old_str not found in file — "str not found, read file first".
