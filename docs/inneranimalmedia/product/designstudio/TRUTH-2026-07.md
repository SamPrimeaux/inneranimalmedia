## BIM wrong-plane root cause (fixed 2026-07-07)

FreeCAD exports **Z-up** source. Pipeline: STL â†’ Blender â†’ **Y-up GLB**.  
Viewport was applying Z-up correction again â†’ model stood on edge.

**Fix:** `glb_up_axis: Y` in placement sidecar + `applySourceOrientation` skips rotate when source Z + GLB Y.

Files: `agentSamSpatial.ts`, `cadPlacement.ts`, placement JSON, `DesignStudioPage.tsx` spawn meta.
