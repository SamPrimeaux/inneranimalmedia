## 2. created_at datetime type — CRITICAL BUG FIXED

Both `agentsam_usage_events.created_at` and `agentsam_command_run.created_at` are `INTEGER NOT NULL DEFAULT unixepoch()` — unix epoch seconds. Confirmed via PRAGMA and live data (~1782430000).

Original bug in `costs_report`: `datetime('now', modifier)` returns TEXT; comparing TEXT against INTEGER epoch silently returns zero rows. Every `costs report` call was broken.

Fix: `unixepoch('now', '-N unit')`. Added `_usage_events_where()` helper with docstring so this doesn't regress.

## 5. Missing executor targets — false positive corrected

85 active `router_type='tool'` commands have `tool_key=NULL`. These are intentional wrangler-proxy commands where `mapped_command` carries the identity. Original check flagged all 85 as broken.
