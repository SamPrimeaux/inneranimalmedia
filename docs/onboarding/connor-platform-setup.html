Before you start

Two paths, one workspace

You have complementary ways to work — pick what fits each task:

Path Best for Needs your VM?
GitHub OAuth Agent file edits, repo browse, ChatGPT MCP tools No
Your VM terminal PowerShell, scripts, git in the dashboard shell Yes
Do not use the shared “Cloud” terminal for your Git identity The platform cloud host (terminal.inneranimalmedia.com) runs under Sam's infrastructure. For Connor git work, always use Start local after connecting your own VM tunnel.
OAuth is enough for most agent work Steps 2 + 3 cover ChatGPT and dashboard agents. Step 5 (GitHub CLI) is only if you want git push from the in-dashboard terminal on your VM.

Link ChatGPT to Agent Sam so tools run scoped to your workspace.

Step 04 · Optional but recommended

Connect your VM / PowerShell terminal

Use your GCP VM (PowerShell) as the dashboard terminal — not the shared cloud host.

On your VM (one-time)

  1. A

    Install and run iam-pty (terminal server on port 3099). Ask Sam for the iam-pty repo if you don't have it yet.

  2. B

    Expose it with a Cloudflare Tunnel so the dashboard reaches it securely (wss://…).

  3. C

    Keep iam-pty and cloudflared running while you use the terminal.

In the dashboard

  1. 1

    Go to Settings → Local terminal (iam-pty) (or the terminal welcome screen).

  2. 2

    Platform: Linux (or Windows) · Shell: PowerShell / pwsh.

  3. 3

    Click Create connection row, paste your tunnel WebSocket URL, then Activate local terminal.

  4. 4

    In the Agent terminal panel, choose Start local — not Cloud.

Isolated workspace path Your files live under /workspace/tenant_connor_mcneely/<your-user-id>/ — separate from other tenants.

Step 05 · Only if you use git in the terminal

GitHub CLI on your VM

Run this on your VM (Linux/bash or WSL). It stores your PAT in ~/.config/iam/github.env only — never in shared platform files.

git clone https://github.com/SamPrimeaux/inneranimalmedia.git cd inneranimalmedia git pull origin main ./scripts/install-terminal-github-cli-connor.sh

When prompted, paste a Personal Access Token for connordmcneely96 (classic PAT with repo scope).

Windows without WSL? Install GitHub CLI manually: winget install GitHub.cli then gh auth login with your PAT.
Identity guard The Connor script rejects tokens for @SamPrimeaux and never installs on Sam's shared GCP VM.
