# Connor Platform Operator Rubric

Scoring:

0 = Not present
1 = Started but not usable
2 = Partially usable with manual work
3 = Usable for internal testing
4 = Production-ready with guardrails
5 = Production-ready, monitored, documented, and tested

Core assignment criteria:

- Correctness: task works as requested.
- Safety and secrets: no secrets exposed; risky actions approval-gated.
- Evidence quality: commands, outputs, screenshots, links, or notes prove completion.
- Conceptual understanding: Connor can explain what happened and how to debug.
- Operator independence: Connor can repeat the workflow with minimal help.

Final readiness criteria:

- Repo operations
- Cloudflare operations
- AI operations
- Integration plan
- Testing and release
- Security and approvals
